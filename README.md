ReqLister
====
 Parses a file and outputs a formatted list of requirements

## Feature List
 - Parse file and converts each line into its own object
 - Ignore empty lines
 - Ignore invalid lines
 - *Handle invalid lines*
 - Format duedate
 - Allow custom duedates
 - Allow no duedate
